# History Date Slider - Руководство по запуску

## Описание проекта

Компонент для отображения временных отрезков с событиями. Реализован с использованием React, TypeScript, SASS, Swiper и GSAP.

## Требования

- **Node.js** >= 16.x
- **npm** >= 8.x

## Установка зависимостей

Перед первым запуском необходимо установить все зависимости проекта:

```bash
npm install
```

Эта команда установит все необходимые пакеты, указанные в `package.json`:

- React и React DOM
- TypeScript и связанные типы
- Webpack и webpack-dev-server
- Swiper для слайдеров
- GSAP для анимаций
- SASS для стилей
- И другие зависимости

## Запуск проекта

### Режим разработки

Для запуска проекта в режиме разработки с hot-reload:

```bash
npm run dev
```

После запуска:

- Проект будет доступен по адресу: `http://localhost:3000`
- Webpack автоматически откроет браузер
- Изменения в коде будут автоматически применяться без перезагрузки страницы

### Сборка для продакшена

Для создания оптимизированной сборки проекта:

```bash
npm run build
```

Собранные файлы будут находиться в папке `dist/`.

### Сборка в режиме разработки (без оптимизаций)

```bash
npm run build:dev
```
